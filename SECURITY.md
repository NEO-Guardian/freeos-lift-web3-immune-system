# Security Policy

## Status

This repository is a **reference implementation** of the N.E.O. Web3 Immune System
for Freeos / Lift style projects on ICP / DFINITY.

Treat it as a **pattern and starting point**, not a finished, audited module.

Before putting real funds behind any implementation based on this repo, projects should:

- Commission an independent security review / audit
- Run extensive simulation, fuzzing and chaos testing
- Start with limited value and monitor behaviour closely
